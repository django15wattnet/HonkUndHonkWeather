Reading and displaying open-meteo.com weather forecast for locations of your choise.

== Description ==
A simple to use shortcode to display the open-meteo.com weather forecast for locations of your choise.

== Installation ==
This section describes how to install the plugin and get it working.
1. Upload [`HonkUndHonkWeather`](https://github.com/django15wattnet/HonkUndHonkWeather/blob/main/honkUndHonkWeather.zip) to the `/wp-content/plugins/` directory

2. Activate the plugin through the 'Plugins' menu in WordPress

== Requirements ==
1. PHP >= 8.2
2. Python >= 3.0
3. Unix-like operating system (Linux, MacOs, BSD, ...)

== How to use ==
Add a shortcode like this example to your post or page:
[honkUndHonkWeather lon=50.72633 lat=1.603076 headline="Boulogne-sur-Mer"]

This first release can display only one forecast per post or page.
